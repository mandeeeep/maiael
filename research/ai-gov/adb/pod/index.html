Getting Started This website loads content from external markdown files. Update the `playlistData` array in the JavaScript section with your file paths. Required folder structure:
audio/*.m4a | transcript/*.md | materials/*.md | feedback/*.md
Note: This website must be hosted on a web server (like GitHub Pages) to load external files. It won't work when opened directly from a local file.
